# 自动推断型类型

> **<sup>句法</sup>**\
> _InferredType_ : `_`

自动推断型类型要求编译器尽可能根据周围可用的信息推断出实际使用类型。它不能用于程序项的签名中。它经常用于泛型参数中：

```rust
let x: Vec<_> = (0..10).collect();
```
